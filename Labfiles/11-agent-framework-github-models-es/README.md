# Laboratorio 11: Flujo de Trabajo de Revisión de Código con Múltiples Agentes

En este laboratorio, explorarás un flujo de trabajo de múltiples agentes para la generación y revisión de código, utilizando `agent-framework` y los modelos de GitHub para la inferencia.

## Instrucciones

1.  **Configura tus variables de entorno.**
    Asegúrate de que tu archivo `.env` en la raíz del repositorio contiene tu `GITHUB_TOKEN`.

2.  **Ejecuta el script.**
    Navega al directorio `Labfiles/11-agent-framework-github-models-es/Python` y ejecuta el siguiente comando:
    ```bash
    python code_review_workflow.py
    ```

3.  **Interactúa con la interfaz de usuario.**
    Se abrirá automáticamente una interfaz de usuario de desarrollo en tu navegador. Puedes usarla para interactuar con el flujo de trabajo.
