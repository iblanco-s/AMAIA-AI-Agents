# Cambios de Temática en los Ejercicios

Este documento registra los cambios de temática realizados en los ejercicios del curso de agentes de IA.

## Módulo 03 - Azure AI Agents SDK Functions

**Temática anterior:**
- Agente de soporte técnico especializado en fabricación aeroespacial
- Problemas con componentes de turbinas, motores, procesos de fabricación
- Ejemplos: microgrietas en palas de turbina, defectos en álabes, problemas en cámaras de combustión
- Ubicaciones: plantas de fabricación en España, Reino Unido, México, EE.UU., Malta, India

**Temática actual:**
- Agente de soporte técnico de TI general
- Problemas con software, sistemas, redes y hardware corporativo
- Ejemplos: errores de autenticación en ERP, problemas de acceso a sistemas, incidencias de red
- Ubicaciones: oficinas en Madrid, Barcelona, Valencia, Sevilla, Bilbao

---

## Módulo 05 - Agent Framework

**Temática anterior:**
- Agente de informes de gastos de operaciones aeroespaciales
- Gastos relacionados con fabricación aeroespacial
- Ejemplos: viajes a plantas de fabricación, materiales de pruebas de turbina, certificaciones de calidad aeroespacial, consultoría técnica aerospace
- Destinos: Planta UK, Planta España, Planta México

**Temática actual:**
- Agente de informes de gastos corporativos generales
- Gastos empresariales de oficina estándar
- Ejemplos: viajes de negocios, reuniones con clientes, material de oficina, cursos de formación, licencias de software
- Destinos: Oficina Barcelona, Oficina Madrid

---

## Módulo 06 - Agent Framework MCP Remote

**Temática anterior:**
- Agente especializado en ingeniería aeroespacial y fabricación aeronáutica
- Consultas sobre documentación técnica aeroespacial
- Ejemplos: mantenimiento predictivo con datos de sensores, documentación técnica aeronáutica
- Nombre del agente: AeroDocsAgent

**Temática actual:**
- Agente consultor técnico general para documentación Microsoft/Azure
- Consultas generales sobre tecnología y servicios Azure
- Ejemplos: configuración de Azure AI Search, Azure Machine Learning, patrones RAG
- Nombre del agente: TechDocsAgent
- **Mejora añadida:** El agente ahora responde en 2-3 líneas máximo para mayor concisión

---

## Módulo 07 - Agent Framework MCP Local Tools

**Temática anterior:**
- Agente de gestión de inventario aeronáutico
- Componentes aeroespaciales y de turbinas
- Ejemplos: álabes de turbina, carcasas de compresor, conjuntos de sello, ejes, revestimientos de combustor
- Servidor MCP: AeroInventory

**Temática actual:**
- Agente de gestión de inventario de productos generales
- Productos de oficina y tecnología
- Ejemplos: portátiles, monitores, teclados, ratones, auriculares, sillas ergonómicas, mesas ajustables
- Servidor MCP: ProductInventory

---

## Módulo 08 - Agent Framework Orchestration

**Temática anterior:**
- Workflow de ingeniería aeroespacial con orquestación secuencial
- Procesamiento de problemas técnicos aeronáuticos
- Ejemplos: vibraciones en motores, fallos en aviónica ARINC 429, refuerzo de estructuras MTOW, trazabilidad de componentes UltraFan
- Agentes: Ingeniero de Sistemas Aeronáuticos, Ingeniero de Aeronavegabilidad, Planificador Técnico Aeroespacial

**Temática actual:**
- Workflow de gestión de proyectos con orquestación secuencial
- Procesamiento de solicitudes y problemas de proyectos generales
- Ejemplos: migración a microservicios, campañas de marketing digital, sistemas de gestión documental
- Agentes: Analista de Requisitos, Gestor de Riesgos, Planificador de Proyectos
