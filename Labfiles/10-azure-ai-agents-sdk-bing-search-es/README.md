# Laboratorio 10: Creación de un Agente de Investigación con Búsqueda de Bing

En este laboratorio, aprenderás a crear un agente de IA que utiliza la Búsqueda de Bing como herramienta para responder a preguntas sobre una amplia variedad de temas.

## Instrucciones

1.  Abre el Jupyter Notebook `Python/bing-search-agent.ipynb`.
2.  Sigue las instrucciones del notebook para crear, ejecutar e interactuar con el agente.
